# Sample token sale

## Requirements
- node v9.10.1
- npm 5.8.0

## Install
- `$ npm install`

## Set path for dependencies usage
- `$ source set_path.sh`

## Migrate
- `$ truffle migrate --network NETWORK_NAME`

## Test
- `$ truffle test --network NETWORK_NAME`

# Local development
Run Ganache:

- `$ ganache-cli --port 7545`

and run tests with:

- `$ truffle test --network development`

or:

- `$ npm run test:development`
